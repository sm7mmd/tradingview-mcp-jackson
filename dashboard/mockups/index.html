Mawjah — 3 design directions

Same data slice (summary tiles · momentum buy-list table · top-pick card), three aesthetics. Open each, compare, then tell me which one (or which parts to mix).

A · Terminal Institutional

Bloomberg-style quant terminal. Maximum density, monospace numbers, hairline grids, sharp corners, phosphor green/red. For a power user who lives in the data.

B · Editorial Quantango-inspired

Premium minimal. Huge display numbers, generous whitespace, one electric accent, restrained color. "Less but bigger" — looks like a funded fintech brand, not a dashboard template.

C · Fintech Polished SaaS

Modern dark SaaS (Linear/Vercel feel). Subtle gradient glow, rounded cards, gradient accents on key numbers, balanced density. Approachable-premium, smooth.

These are throwaway mockups in dashboard/mockups/ — the live app is untouched.
